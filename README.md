## Development / How to run the Small CRM  Project

To get a local copy of the code, clone it using git:

```
git clone https://github.com/Mr2RoB0T/small_CRM_project
cd small_CRM_project
```

```
Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
```

```
Open PHPMyAdmin (http://localhost/phpmyadmin)

Create a database with name crm
```

```
Import crm.sql file(given inside the zip package in SQL file folder)

Run the script http://localhost/crm (frontend)

Run the script http://localhost/crm/admin 
```

```
This project is a lightweight Customer Relationship Management (CRM) system designed to 
help businesses manage their interactions with current and potential customers.
It provides a straightforward and intuitive interface for managing contacts, tracking interactions, 
and analyzing customer data.
```

```
FeaturesContact Management: 
Store and manage detailed information about your customers.Interaction Tracking: Log communications and interactions with customers to maintain a history of engagement.
Lead Management: Track potential customers and sales opportunities from initial contact to conversion.
Task Management: Assign and manage tasks related to customer interactions to ensure timely follow-up.
Reporting and Analytics: Generate basic reports to gain insights into customer behavior and interaction patterns.
```

```
UsageAdd Contacts: Navigate to the Contacts section and click "Add Contact" to enter new customer information.
Log Interactions: In the Interactions section, click "Log Interaction" to record communications with customers.
Manage Leads: Go to the Leads section to add and update information about potential customers.
Assign Tasks: Use the Tasks section to assign and track tasks related to customer interactions.
Generate Reports: Access the Reports section to create and view basic analytics on customer data
```
